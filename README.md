# 漂流瓶计划前后端集成

这个仓库包含漂流瓶计划项目的前后端集成代码，用于实现前端界面与后端API的串联。

## 项目结构

```
├── FE                  # 前端代码
│   ├── common          # 公共组件和工具
│   ├── images          # 图片资源
│   ├── index           # 首页
│   ├── invitation      # 邀请页面
│   ├── reservation     # 预约页面
│   └── task            # 任务页面
└── service             # 后端服务代码
```

## 技术栈

- 前端：原生 JavaScript, HTML, CSS
- 后端：Node.js, Express, MongoDB
